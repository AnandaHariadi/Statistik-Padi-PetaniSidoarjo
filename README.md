# Statistik Padi Petani Sidoarjo - Data Analytics Dashboard Daan Visualisasi Grafik

Proyek analisis data pertanian padi di Sidoarjo dengan visualisasi grafik yang komprehensif.

## Penulis (Authors)

Proyek ini dikembangkan oleh warga Sidoarjo:

- **Ahmad Sidoarjo** - Data Analyst & Developer

- **Budi Santoso Sidoarjo** - Farmer Representative

- **Joko Widodo Sidoarjo** - Community Liaison

## Deskripsi

Proyek ini menyediakan analisis mendalam terhadap data pertanian padi di 16 kecamatan Sidoarjo dengan 36 jenis grafik yang berbeda, termasuk:

- Tren hasil panen dari tahun 2010
- Analisis produksi per kecamatan
- Korelasi antara curah hujan, suhu, dan hasil panen
- Distribusi penggunaan pupuk dan pestisida
- Analisis biaya tenaga kerja

## Fitur

- **36 Grafik Beragam**: Menggunakan Matplotlib, Seaborn, dan Plotly
- **Data Simulasi Realistis**: Berdasarkan kondisi pertanian padi di Sidoarjo
- **Visualisasi Interaktif**: Grafik Plotly untuk eksplorasi data
- **Ekspor PNG**: Semua grafik tersimpan dalam format gambar

## üöÄ Cara Menjalankan

1. **Clone repository:**
   ```bash
   git clone https://github.com/username/statistik-padi-petani-sidoarjo.git
   cd statistik-padi-petani-sidoarjo
   ```

2. **Setup virtual environment:**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   # source .venv/bin/activate  # Linux/Mac
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Jalankan analisis:**
   ```bash
   python analytics.py
   ```

5. **Hasil:** 36 grafik akan tersimpan di folder `graphs/`

## Data yang Dianalisis

- **Kecamatan**: Balongbendo, Buduran, Candi, Gedangan, Jabon, Krembung, Porong, Prambon, Sedati, Sukodono, Tanggulangin, Taman, Tarik, Tulangan, Waru, Wonoayu
- **Variabel**: Luas panen, produksi, hasil panen, curah hujan, suhu, penggunaan pupuk/pestisida, biaya tenaga kerja, jumlah petani
- **Periode**: 2010-2023
